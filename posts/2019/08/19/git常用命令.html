# tag管理

## 删除tag
git tag -d v0.0.1

git push origin --delete tag <tagname>

# 合并不相干的历史

如果本地新建了一个仓库且添加了一些文件，远程也新建了一个仓库且添加了一些文件，此时合并会提示：fatal: refusing to merge unrelated histories，意思是两个不相关的仓库不能直接合并

解决办法：

#通过参数指定允许相关的仓库进行合并
git pull --allow-unrelated-histories
